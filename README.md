# Instagram E-Commerce Integration

## Project Overview
This project explores integrating e-commerce functionality within Instagram to provide a seamless shopping experience.

## Problem Statement
Users often leave Instagram to complete purchases. This design aims to enable product discovery and purchasing directly from the platform.

## User Flow
Instagram Feed
→ Cart Icon
→ Shopping Home
→ Product Details
→ Cart
→ Checkout

## Tools Used
- Figma
- Wireframing
- Prototyping
- UI/UX Design

## Key Features
- Integrated shopping experience
- Product browsing
- Shopping cart management
- Streamlined checkout flow
- User-centered interface design

## Screens
- Instagram Feed
- Shopping Home
- Product Details
- Cart
- Checkout
